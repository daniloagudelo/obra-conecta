# OBRA-CONECTA 
Danilo Agudelo 

Sistema web desarrollado para conectar clientes con trabajadores del sector de construcción, remodelación y servicios del hogar de forma rápida, organizada y segura.

## Descripción del proyecto

Obra Conecta nace como una solución para facilitar la búsqueda de mano de obra confiable en áreas como:

- Construcción
- Electricidad
- Plomería
- Pintura
- Remodelaciones
- Carpintería
- Servicios generales

La plataforma permite que los usuarios publiquen necesidades o encuentren trabajadores según el servicio requerido, optimizando tiempo y mejorando la comunicación entre ambas partes.

## Objetivo

Crear una plataforma moderna, intuitiva y funcional que facilite la conexión entre clientes y trabajadores independientes del sector de obras y mantenimiento.

## Características principales

- Registro e inicio de sesión de usuarios
- Publicación de servicios
- Búsqueda de trabajadores
- Perfil de trabajadores
- Sistema de contacto
- Diseño responsive
- Interfaz moderna y fácil de usar
